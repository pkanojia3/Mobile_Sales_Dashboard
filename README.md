# Mobile Sales Dashboard (Power BI)
This project presents an interactive Power BI dashboard built to analyze mobile phone sales performance across different cities, 
time periods, and customer segments. The dashboard provides insights into sales trends, customer ratings, payment behavior, and top-performing mobile models.

## Dataset
Mobile Sales Data from Skill course


## Insights
- Delhi shows the highest sales concentration.
- Metro cities dominate sales
- Tier-2 cities contribute but with smaller sales volumes.
- Demand for mobile devices is consistent throughout the year, with moderate seasonal fluctuations.
- Majority of customers give high ratings (4–5 stars).
